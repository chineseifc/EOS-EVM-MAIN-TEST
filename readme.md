# 什么是EOSEVM网?（what is eosevm network?）:
EOSEVM是EOS网络的二层，它支持以太坊虚拟机，所以叫它EVM，它以EOS作为手续费，其他和以太坊是一样的。

# 如何将主网的EOS跨链到EOSEvm网络?（How to cross from MainNet to EvmNet?）：

1、使用官方桥
官方桥地址：https://bridge.evm.eosnetwork.com/

2、使用交易所直接提币
